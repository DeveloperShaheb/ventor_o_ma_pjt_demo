# Aurelia Vatican Travel

A responsive luxury Vatican Museums ticketing and trip-planning website built with plain HTML, CSS, and JavaScript.

## Pages
Home, Tickets, Ticket Detail, Booking, Checkout, Confirmation, About, Visitor Guide, Accessibility, FAQ, Contact, Blog, Policies, and Admin Dashboard.

## Notes
- Independent platform disclaimer is shown in the header, footer, and checkout flow.
- Ticket prices, policies, homepage content, coupons, FAQs, and inventory can be edited from the admin dashboard.
- Default visitor information is seeded from official Vatican Museums pages and kept editable.

## Run locally
Open `index.html` in a browser or serve the folder with any static server.
